# 90DNS & DNS MITM tester
A simple switch homebrew that does DNS resolution to see if we reach Nintendo's servers. **This works with DNS MITM** and 90DNS.

It will tell you if you're successfully blocking or if there are issues. This is safe to run on SysNAND provided you dont make any changes when doing so.
